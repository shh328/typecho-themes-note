# Typecho默认主题修改 - note

基于默认主题进行了改造。

关于主题命名，目的就是记录，既然旨在 `记录` ，那 `note` 一词自然是再贴切不过了，`note` 之名便应运而生。

具体修改内容如下：

* **大字报模式**
    此为主题修改的核心之一，对字体大小的调整，并对整体宽度进行了少量加宽。考虑到屏幕尺寸日益增长的趋势，以及本人日益增长的年龄看大字眼睛会更舒服，增大了整体字体。无论是正文内容，还是导航栏，均有所调整。
* **文章列表模式**
   首页、归档等列表页面改为了列表模式，只留下标题。
* **霞鹜文楷字体**
   在字体的选择上，特意选择了 [霞鹜文楷](https://github.com/lxgw/LxgwWenKai) 。虽然这款字体基于某日文字体衍生，但其兼有仿宋和楷体的特点，另外字体命名对我的眼缘 `落霞与孤鹜齐飞，秋水共长天一色`。
* **深色模式**
   最后，便是加入了深色模式。仅仅是希望能对眼部疲劳带来一定的缓解。
   `深色模式是让AI写的。`
